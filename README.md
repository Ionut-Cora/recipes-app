# Recipe App

## For this App I used:
- HTML
- JavaScript
- SASS
- Free Edamam API

## App description:
This is an app for Healthy and Nutritious food receipes. The user can write in the search bar what ingredient prefer, or what dish prefer and he can find what receipe he want.
